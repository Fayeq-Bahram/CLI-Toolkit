
# Fayeq-Bahram-CLI-Toolkit-W1
# Description
* CLI Toolkit is a set of simple commands built with Node.js to analyze and manage CSV files. It can show statistics filter data, sort records and export results.
 This CLI toolkit enable developers to:
  * Automate workflow.
  * read and show statistics of the very data
  * sort and export data to a new file
  
# Sample Basic Commands
 * node stats index.js
 * Node index.js sort data/data.csv price
 * Node index.js export data/data.csv oupts.csv
   
# Key Features
  * Customizable commands
  * fast and lightweight
  * supports plugins extensions
  * my toolkit simplifies reading/sorting and reducing errors
